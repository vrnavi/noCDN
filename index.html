About this site.

This site was made to aggregate links to files hosted on CDN-backed services like Discord, to be able to download such files without signing up for these services. You can see the relevant blog post here.

This site takes heavy inspiration from based.cooking. Do check them out.

It’s easy to contribute!

If you have a link to a file hosted on a CDN-backed service, please submit it on Github. The rules for such are posted there.
